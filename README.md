# 📊 Data Analysis Projects Collection

Welcome to my **Data Analysis Projects Repository**. This repository contains multiple real-world data analysis projects developed using **Python** and **Pandas**. Each project focuses on exploring datasets, performing data cleaning, filtering, grouping, and extracting meaningful insights.

---

## 🚀 Projects Included

### 1. 🎓 Admission Prediction Analysis
**Dataset:** Admission_predict.csv

#### Description
This project analyzes graduate admission data based on various student performance metrics such as GRE, TOEFL, CGPA, University Rating, SOP, LOR, Research experience, and more.

#### Key Analysis
- Filter students based on GRE, TOEFL, CGPA
- Research vs Non-Research applicants
- University Rating analysis
- Admission chance exploration
- Data filtering and querying

**Skills Used**
- Pandas
- Data Filtering
- Data Exploration
- Boolean Indexing

---

### 2. 🏠 Airbnb Data Analysis
**Dataset:** Airbnb.csv

#### Description
This project analyzes Airbnb listings to understand pricing, room types, locations, and other property details.

#### Key Analysis
- Display first and last records
- Explore dataset dimensions
- Analyze room types
- Price analysis
- Listing information
- Column selection

**Skills Used**
- Pandas
- Data Cleaning
- Exploratory Data Analysis (EDA)

---

### 3. 🏡 House Price Analysis

#### Key Analysis

- Location-based filtering
- Bedroom extraction
- Property comparison
- Price analysis

**Skills Used**
- Pandas

- Feature Engineering
- Data Filtering

# ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Ramesh-sirvi/Data-Analysis-Projects.git
```

2. Install required libraries

```bash
pip install pandas numpy jupyter
```

3. Open Jupyter Notebook

```bash
jupyter notebook
```

4. Open any project notebook and run the cells.
